# Jogo Gourmet

## Para clonar
```
git clone https://github.com/dvriesman/meujogogourmetx.git
```

## Para compilar no Linux
```
cd meujogogourmetx
./gradlew build
```

## Para compilar no Windows
```
cd meujogogourmetx
gradlew build
```


## Para executar no Linux
```
cd build/libs
java -jar MeuJogoGourmetX.jar
```


## Para executar no Windows
```
cd build\libs
java -jar MeuJogoGourmetX.jar
```
